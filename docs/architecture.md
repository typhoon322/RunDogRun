## 架构特点

- **单向数据流**: AKShare → 策略 → 回测 → 监控 → 日报 → UI
- **无实时依赖**: 所有计算基于本地缓存
- **完全可重放**: 快照冻结确保回测不变
- **UI 与策略解耦**: Streamlit 只读取 JSON，不参与计算

## 关键设计

1. **Snapshot 机制**: 每日冻结数据，回测绑定快照日期
2. **Pipeline 日志**: 每步记录，白盒可观测
3. **策略门控**: TRADE/CAUTION/STOP 三态
4. **漂移检测**: 历史 vs 近期对比，量化退化程度
